SDG 13.3 提升气候变化教育与能力建设

Rocklea（Brisbane）社区气候韧性系统地图

本项目聚焦联合国可持续发展目标 13（气候行动）之目标 13.3： 通过交互式系统地图探索如何提高气候素养、优化早期预警、增强社区准备度，并以人机交互视角识别机会点。

问题陈述

极端高温与火险上升，但用户对风险的理解、信任与行动之间存在断裂。

工作假设

当预警清晰、可信且可达，配合在地化学习与演练，社区韧性将随之提升。

交付

系统地图（CLD/SFD）、网站与分析文档、方法与工具说明。

更新：2025-08-19

系统边界与目的

边界涵盖：数据采集（传感/遥感）→ 风险建模 → 预警发布 → 用户理解 → 行动 → 影响评估 → 数据回流。 目的在于减少健康与财产损失，提升自主准备与集体互助。

与 SDG 的对齐

直接对齐 SDG 13.3：教育与能力建设；并与 SDG 3（健康）、SDG 11（可持续城市） 产生协同效应。
